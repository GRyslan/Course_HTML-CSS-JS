Chicken

Hi there! It is Bootstrap 5.1.3, not 3.X.X (which used in this course).

Jquery dependency removed.

Default display flex ,not block,

navbar-default --> navbar dark(light),

visibile-xs --> d-sm-none,

navbare-toggle --> navbar-toggler,

> data-toggle(target) --> data-bs-toggle(target).

Three span with class icon-bar --> one span with class navbar-toggler-icon,

xs breakpoint changed from <750px to <576px,

sm breakpoint changed from ≥768px to ≥576px,

md breakpoint changed from ≥992px to ≥768px,

fw-bold=font-weight:bold;

text-white=color:white;

bg-black(dark,secondary) = background-color:black(.etc);

border-5 = border-width:5px;

border-dark = background-color:some-color-code;

shadow-none - hide shadow in button(on :focus),

text-center = text-align:center;

text-end = text-align:right(end)

m(p)(b)(t)-2 = margin(padding)(-bottom)(-top):$spacer * .5;(bootstrap default)
